## Inventory Management System

This project is a simple yet powerful Inventory Management System designed to help track, manage, and organize products efficiently. It supports adding, updating, and deleting items, monitoring stock levels, and generating reports for better decision-making. Built with scalability in mind, it ensures smooth handling of inventory for businesses of small sizes.


## Features
- Track all the inventory items
- Get stock status information
- Manage inventory and allocation
- PDF generation
- Stock Graphs
- Dashboard


## Tech Stack
- Frontend: HTML, CSS, JS ,Bootstrap
- Backend: PHP
- Database: MySQL
